# rikulife

人工生命を作ってみたい！！

- 50x50のフィールドに生命を放つ。餌がランダムでスポーン。
- 各生命は状態として、HPと色（RGB）を持ってる。
  - 色は適宜変えられる。
- 行動としては、上下左右に進む、何もせずに待機、攻撃、他の生命へHPを分け与えることができる。

- これをneural evolutionでやる。
- 周囲5x5に餌があるか、壁（範囲外）があるか、他の個体がいるか（いた場合は色を入力）、自分のHPを入力にする。
- 出力は、行動と色。

## 実装

`ndarray`で自前でNNを実装。

- `burn`や`tch-rs`は今回の用途には重い？

- 更新順はランダム
- ユニークなIDを割り当てて、記録や更新に使う。

## 記録

- 統計データ
  - CSV、JSONなどで保存する。
  - 全体の傾向を見る用。Pythonなどでグラフにする。
  - データ
    - Step: 経過ターン数
    - Population: 生存数
    - FoodCount: フィールドにある餌の総数
    - MaxGen: いま生きている中で最大の世代数（第何世代か）
    - AvgEnergy: 全員の平均エネルギー
    - ActionRate: 行動の内訳（攻撃回数、移動回数、回復回数の割合）

- 家系図の記録
  - 生まれたほうのID
  - 産んだほうのID
  - そのときの世界のステップ
  - 何世代目か


## 初期パラメータ案

```csv
パラメータ名,設定値（例）,意味
MAX体力,100,これ以上は増えない上限
初期体力,50,生まれた時の体力
行動コスト,1,移動・待機・攻撃・回復で減る量
餌の回復量,15 (15%),1個食べた時の回復量
繁殖条件,体力100到達,これになった瞬間に強制執行
繁殖コスト,50 (50%),親から引かれる体力
攻撃威力,20,攻撃した時に相手から奪う/消す量
回復威力,20,回復した時に相手にあげる量
```
